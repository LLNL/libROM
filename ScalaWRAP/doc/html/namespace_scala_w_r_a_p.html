◆ svd() [1/2]

SVDInfo ScalaWRAP::svd ( ScalaMat::Submatrix &  A,
bool  wantu = true,
bool  wantvt = true 
)

Compute the SVD of A.

Computes the singular value decomposition of a (submatrix of) a distributed A. The results are returned in the SVDInfo struct. This routine should only ever fail due to numerical issues, since every matrix has an SVD.

Parameters
[in,out]AA Submatrix. The SVD of A is computed and the data stored in A is destroyed in the process.
[in]wantu(Default: true) Set to false if you don't need the matrix U of left singular vectors.
[in]wantvt(Default: true) Set to false if you don't need the matrix Vt of transposed right singular vectors.
Returns
An object of type SVDInfo. This contains pointers to U and Vt, if they were requested, and the singular values in a std::vector as well as the return code from PDGESVD. If A is m x n and SIZE = min(m, n), U is m x SIZE, there are SIZE singular values, and Vt is SIZE x n.
